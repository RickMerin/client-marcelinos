# 🗺️ InteractiveMap Component (React + Leaflet)

A reusable React component for displaying an **interactive map** of **Marcelino’s Place** in **Hilongos, Leyte**, using **React-Leaflet** and **OpenStreetMap** tiles.  
It includes a custom map marker icon and popup information.

---

## 🚀 Features

- Interactive zoom and pan support  
- Custom pin icon (PNG or SVG)  
- Popup with place name and address  
- Fully responsive — scales with container size  
- Built on **React Leaflet + Leaflet**

---

### 1. Install dependencies
